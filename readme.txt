這個repo是用來練習github
請大家使用pull request的方式新增一個任意檔案到自己名字的branch
然後我會將大家的branch merge進main
1. 準備好一個空folder
2. 用vscode開啟
3. fork共用repo到遠端repo
4. clone 遠端repo 到 local repo
5. 在local repo新增一個branch，名字是自己英文名
6. 在新增的branch中加入一個檔案，檔案名為:英文名.txt, 並commit進去
7. push此branch到遠端repo(tony->origin/tony)
8. pull request到共用repo(遠端repo/tony->共用repo/tony)
